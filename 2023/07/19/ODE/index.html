Ordinary Differential Equation (ODE)

The defination of ODE

An ODE is an equation in which the unknown quantity is a function, and it also involves derivatives of the unknown function.

For example, the forced spring-mass system:

d2x(t)dt2+γdx(t)dt+v2x(t)=w(t).(1)\frac{d^2 x(t)}{d t^2} + \gamma \frac{d x(t)}{dt} + v^2 x(t) = w(t). \qquad (1) dt2d2x(t)​+γdtdx(t)​+v2x(t)=w(t).(1)

In this equation:

  • vvv and γ\gammaγ are constants that determine the resonant angular velocity
    and damping of the spring.

  • w(t)w(t)w(t) is a given function that may or may not depend on time.

  • position variable xxx is called dependent variable.

  • time ttt is called independent variable.

  • This equation is second order. It contains the second derivative and doesn’t have higher-order terms.

  • This equation is linear. x(t)x(t)x(t) is linearly. There is no terms like x2(t),log⁡x(t)x^2(t), \log x(t)x2(t),logx(t)…

  • This equation is inhomogeneous. Because it contains forcing term w(t)w(t)w(t).

The solution to ODE

It can be divided to two categories:

  • particular solution: a function that satisfies the differential equation and does not contain any arbitrary constants.

  • general solution: a function that satisfies the differential equation and contains free constants.

To exactly solve the differential equation, it is necessary to combine the general
solution with some initial conditions (i.e., x(t0)x(t_0)x(t0​), dx(t)dt∣t0\frac{d x(t)}{dt} |_{t_0}dtdx(t)​∣t0​​) or some other (boundary) conditions of the differential equation.

Linear ODEs

Equation (3) is also a special case of the linear differential equations:

dx⃗(t)dt=F(t)x⃗(t)+Lw⃗(t).\frac{d \vec{x}(t)}{dt} = \mathbf{F}(t) \vec{x}(t) + \mathbf{L} \vec{w}(t). dtdx(t)​=F(t)x(t)+Lw(t).

where F(t)\mathbf{F}(t)F(t) is a matrix-valued function of time. L\mathbf{L}L is a matrix. w⃗(t)\vec{w}(t)w(t) is a vector-valued function of time.

  • homogeneous: the equation is homogeneous if the forcing function w⃗(t)\vec{w}(t)w(t) is zero for all ttt.
  • time-invariant: the equation is time-invariant if F(t)\mathbf{F}(t)F(t) is constant for all ttt.

In the following sections, we first start with the simple scalar linear time-invariant homogeneous differential equation with fixed initial condition at t=0t = 0t=0. Then, we will consider the multidimensional generalization of this equation. Besides, we also consider the linear time-invariant inhomogeneous differential equations. Finally, we will consider more general differential equations.

sX(s)−x(0)=FX(s)+LW(s).s X(s) - x(0) = \mathbf{F} X(s) + \mathbf{L} W(s). sX(s)−x(0)=FX(s)+LW(s).

Then,

X(s)=[sI−F]−1x(0)+[sI−F]−1∗LW(s).(17)X(s) = [s\mathbf{I}-\mathbf{F}]^{-1} x(0) + [s\mathbf{I}-\mathbf{F}]^{-1} \ast \mathbf{L} W(s). \qquad (17) X(s)=[sI−F]−1x(0)+[sI−F]−1∗LW(s).(17)

Compare the solution (17) with solution (8), we can obtain

L−1[(sI−F)−1]=exp⁡(Ft)\mathcal{L}^{-1}[(s\mathbf{I}-\mathbf{F})^{-1}] = \exp(\mathbf{F}t) L−1[(sI−F)−1]=exp(Ft)

for t≥0t \geq 0t≥0.

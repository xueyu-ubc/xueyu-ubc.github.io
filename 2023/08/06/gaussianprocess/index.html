Gaussian processes

Gaussian processes (GP) generalize Gaussian vectors to infinite dimensions.

Definition. X(t)X(t)X(t) is a GP if any linear combination of values X(t)X(t)X(t) is Gaussian.
That is, for arbitrary n>0n > 0n>0, times t1,...,tnt_1, ..., t_nt1​,...,tn​ and constants a1,...,ana_1, ..., a_na1​,...,an​, Y=a1X(t1)+a2X(t2)+...+anX(tn)Y = a_1 X(t_1) + a_2 X(t_2) + ... + a_n X(t_n)Y=a1​X(t1​)+a2​X(t2​)+...+an​X(tn​) is Gaussian distributed.

  • Time index ttt can be continuous or discrete.

  • Any linear functional of X(t)X(t)X(t) is Gaussian distributed.
    For example, the integral Y=∫t1t2X(t)dtY = \int_{t_1}^{t_2} X(t) \text{d}tY=∫t1​t2​​X(t)dt is Gaussian distributed.

The jointly pdf of $X(t_1),..., X(t_n)$ is $N([\mu(t_1), ..., \mu(t_n)]^T, C(t_1,..., t_n))$. ## Mean value and autocorrelation functions To specify a Gaussian process, we only need to specify: - Mean value function: $\mu(t) = \mathbb{E}[x(t)]$. - Autocorrelation function (symmetric): $R(t_1, t_2) = \mathbb{E}[X(t_1) X(t_2)]$. The autocovariance $C(t_1, t_2) = R(t_1, t_2) - \mu(t_1) \mu (t_2)$. More general, we consider GP with $\mu(t) = 0$. [define new process $Y(t) = X(t) - \mu(t)$]. p(x_t) = \frac{1}{\sqrt{2\pi (R(t,t) - \mu^2(t))}} \exp(- \frac{(x_t - \mu(t))^2}{2(R(t,t) - \mu^2(t))}).

## Conditional probabilities in a GP Consider a zero-mean GP $X(t)$, two times $t_1$ and $t_2$. The covariance matrix is

The conditional pdf of $X(t_1)$ given $X(t_2)$ is

p_{X(t_1)| X(t_2)}(x_{t_1} | x_{t_2}) = \frac{p(x_{t_1}, x_{t_2})}{p(x_{t_2})}. \qquad (1)

Brownian motion with drift (BMD)

For Brownian motion, it is an unbiased random walk. Walker steps right or left with the same probability 1/21/21/2 for each direction (one dimension).

For BMD, it is a biased random walk. Walker steps right or left with different probs.

For example, consider time interval hhh, step size σh\sigma \sqrt{h}σh​,

p(X(t+h)=x+σh∣X(t)=x)=12(1+μσh).p(X(t+h) = x + \sigma \sqrt{h} | X(t) = x) = \frac{1}{2} (1 + \frac{\mu}{\sigma} \sqrt{h}). p(X(t+h)=x+σh​∣X(t)=x)=21​(1+σμ​h​).

p(X(t+h)=x−σh∣X(t)=x)=12(1−μσh).p(X(t+h) = x - \sigma \sqrt{h} | X(t) = x) = \frac{1}{2} (1 - \frac{\mu}{\sigma} \sqrt{h}). p(X(t+h)=x−σh​∣X(t)=x)=21​(1−σμ​h​).

  • μ>0\mu > 0μ>0, biased to the right. μ<0\mu < 0μ<0, biased to the left.

  • hhh needs to be small enough to make ∣μσh∣≤1|\frac{\mu}{\sigma} \sqrt{h} | \leq 1∣σμ​h​∣≤1.

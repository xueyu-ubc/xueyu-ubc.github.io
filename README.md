# Xue Yu's Blog

A personal research blog built with Hexo and published through GitHub Pages.

The site collects reading notes, paper summaries, technical reflections, and course notes around modern machine learning and AI systems.

## Scope

- Foundation models, LLM reasoning, transformers, VLMs, and model acceleration
- GUI agents, multi-agent systems, trustworthy agents, and agent frameworks
- World models, embodied AI, diffusion models, and generative modeling
- Reinforcement learning, optimization, stochastic processes, and statistics
- Federated learning, distributed learning, and selected research paper reviews
- Personal learning notes from courses and technical references

## Site

- Blog: https://xueyu-ubc.github.io/
- Repository: https://github.com/xueyu-ubc/xueyu-ubc.github.io

- Hexo static site generator
